## 📁 Directory Structure

| Directory     | Purpose                          |
|---------------|----------------------------------|
| `/home`       | User home directories            |
| `/etc`        | Configuration files              |
| `/bin`        | Essential system binaries        |
| `/var/log`    | System and service logs          |
| `/tmp`        | Temporary files                  |
